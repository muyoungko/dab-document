# Google Login

## Settting Firebase First
Firebase Console > Authentication
Turn on the Google Login


## Android
Download google-services.json
Go to google cloud
Use auth key Web client (auto created by Google Service)
Set project setting Google App Key

## iOS
Download GoogleService-Info.plist
Go to google cloud
Use auth key of OS client for ... (auto created by Google Service)
Set project setting Google App Key for IOS
