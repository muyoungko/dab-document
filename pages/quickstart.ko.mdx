# Quick Start

Devil App Builder(DAB)로 첫 번째 모바일 앱을 만들어보세요!

## 📋 준비

### 시스템 요구사항
- **운영체제**: Only Mac OS X 지원

### 필수 소프트웨어
- **DAB 계정**: [Devil App Builder](https://www.devil-app-builder.com) 사이트 가입
- **Sketch 앱**: [Sketch 공식 사이트](https://www.sketch.com/)에서 다운로드

## 🚀 신규 프로젝트 생성

1. [프로젝트 생성 페이지](https://www.devil-app-builder.com/project/new)로 이동
2. **Project Unique Name** 란에 적절한 프로젝트 이름 입력
3. **Choose Template** > **Blank sketch** 선택
   `템플릿 스캐치로 시작합니다`
4. **Create** 버튼 클릭

![프로젝트 생성 화면](/image/project-new.png)

> 💡 **팁**: 프로젝트 이름은 영문, 숫자, 하이픈(-)만 사용 가능합니다.

## 📱 템플릿 스캐치 다운로드

1. 좌측 메뉴에서 **APP > Sketch** > 다운로드 아이콘 클릭
2. 다운받은 `.sketch` 파일을 Sketch 앱으로 열기

![스케치 목록 화면](/image/sketch-list.png)
![스케치 파일 열기](/image/sketch.png)

## 🔌 스캐치 플러그인 다운로드

Devil App Builder와 Sketch를 연동하는 플러그인을 설치합니다.

### 플러그인 설치
1. [Devil 플러그인 다운로드](https://img.deavil.com/plugin/devil_plugin.sketchplugin.zip)
2. 다운로드된 `.sketchplugin` 파일을 더블클릭하여 설치
3. Sketch 앱 재시작

![플러그인 설치](/image/sketch-plugin.png)

### 플러그인 사용법
- **메뉴**: `Plugin > Devil App Builder > Upload Sketch`
- **단축키**: 자주 사용하는 단축키 설정 권장
- **업로드**: 스케치 수정 후 플러그인으로 업로드하면 앱에 즉시 반영

## 🎯 첫 화면으로 지정

1. 좌측의 **APP > Screen > first** 메뉴 클릭
2. **Critical Setting**에서 **Start Screen** 체크 후 저장

![시작 화면 설정](/image/screen-setting.png)

> ⚠️ **중요**: Start Screen이 설정되지 않으면 앱이 실행되지 않습니다.

## 📲 DAB 앱에서 확인

1. 휴대폰에서 **Devil App Builder** 앱 다운로드
2. 같은 아이디로 로그인 후 해당 프로젝트에서 **Start** 클릭
3. 첫 화면 확인

### 앱 동기화 확인
- 앱의 우측 하단 **Devil 아이콘** 클릭 시 최신 스케치로 리로드
- 스케치 수정 후 플러그인으로 업로드하면 앱에서 즉시 확인 가능

## ➕ 앱 화면 추가

### 스케치에서 새 화면 만들기
1. 스케치에서 **Frame**을 추가합니다. 이름은 `main`으로 합니다
2. **화면사이즈**: `375x820` 권장 (iPhone 12/13/14 기준)
3. 플러그인을 이용해 스캐치 업로드

![새 화면 추가](/image/sketch-main-add.png)

### DAB에서 화면 등록
1. 좌측 메뉴 **Screen > New Screen**에서 **Screen Name Main**이 추가된 것을 확인
2. **Create** 버튼 클릭하여 신규 화면으로 추가

![새 화면 등록](/image/new-screen.png)

## 🖱️ 클릭 처리

### 화면 간 이동 구현
1. 좌측 메뉴 **Screen > first** 화면으로 이동
2. **first** 화면에서 화면 터치시 **main**화면으로 이동하기
3. 화면 중앙에 위치한 **Block Rule** 목록에서 **Click** 선택
4. **New Rule** 선택 후 **Node** 부분에 `"My First App"` 입력
5. **Click Action**에 다음 코드 추가:

```js filename="Click Action"
Jevil.go('main')
```

6. **Add New** 클릭

![클릭 이벤트 설정](/image/click-mapping.png)

> 📚 **참고**: 더 자세한 화면 이동 방법은 [Jevil.go 문서](/ko/mobileapp/jevil/screenmove#jevilgo)를 확인하세요.

## 📝 텍스트 처리

1. 스캐치에 "myname" 텍스트 노드 추가 후 업로드

![텍스트 노드 추가](/image/myname-text.png)

2. **APP > Screen > main** 화면으로 이동하여, **Javascript Editor**에 다음 코드 입력:

```js filename="Javascript"
data.name1 = 'Jason Ko'
```

> 💡 **참고**: `data` 변수는 예약어입니다.

3. "Text Mapping" 블럭 룰을 추가하여 `data.name1`과 `myname` 노드를 연결:
   - **Node**: `"myname"`
   - **Data Json Path**: `name1`

> 📝 **설명**: `name1`은 `data.name1`을 의미하며, `data` 변수는 Screen 단위로 내장되어 선언됩니다.

4. DAB 앱에서 **Devil 아이콘** 클릭하여 리로드한 후 해당 내용 적용 확인

### 네트워크 API 처리
- 외부 API 연동 및 데이터 처리

### 리스트 처리
- 동적 리스트 표시 및 스크롤 처리

## ✅ 완료!

축하합니다! 기본적인 Devil App Builder 앱이 완성되었습니다.
