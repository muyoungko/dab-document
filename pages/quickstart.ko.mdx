# Quick Start

### 준비
- Only Mac OS X 지원
- DAB 사이트 가입
- https://www.devil-app-builder.com 
- 스케치 어플리케이션 다운로드
- https://www.sketch.com/ 

### 신규 프로젝트 생성
 - https://www.devil-app-builder.com/project/new
 ![Quickstart 이미지](/image/project-new.png)
 - Project Unique Name 란에 적절한 프로젝트 이름 입력
 - Choose Template > Blank sketch 선택
 - Create 버튼 클릭

### 템플릿 스캐치 다운로드
 - 좌측 메뉴에서 APP > Sketch > 다운로드 아이콘 클릭
 ![Quickstart 이미지](/image/sketch-list.png)
 - 다운받은 스캐치 열기
 ![Quickstart 이미지](/image/sketch.png)


### 스캐치 플러그인 다운로드

스캐치 플러그인을 다운받고 설치한다. Sketch 를 앱과 동기화하는 데 사용한다

https://img.deavil.com/plugin/devil_plugin.sketchplugin.zip

![Quickstart 이미지](/image/sketch-plugin.png)

Sketch를 동기화하는 버튼은 Plugin > Devil App Builder > Upload Sketch 이며, 단축키를 이용하여 자주 업로드한다

### 첫 화면으로 지정
- 좌측의 APP > Screen > first 메뉴 클릭
- Critical Setting에서 Start Screen 체크 후 저장
![Quickstart 이미지](/image/screen-setting.png)

### DAB 앱에서 확인
- 휴대폰에서 Devil App Builder 앱 다운로드
- 같은 아이디로 로그인 후 Start 클릭 
- 첫화면 확인

### 스케치 수정 후 업로드
- 스캐치의 첫 화면의 디자인을 수정해본다 
- 플러그인을 이용해 업로드한다
- 앱의 우측 하단 데빌 아이콘을 클릭하면 동기화한 스캐치로 리로드되며, 이를 통해 스캐치 변경을 확인한다

### 앱 화면 추가
- 스캐치에서 Frame을 추가한다. 이름은 main으로 한다. 화면사이즈는 375x820을 권장
![Quickstart 이미지](/image/sketch-main-add.png)
- 플러그인을 이용해 스캐치 업로드를 한다
- 좌측 메뉴 > Screen > New Screen 에 Screen Name Main이 추가된 것을 확인하며 
![Quickstart 이미지](/image/new-screen.png)
- Create 버튼 클릭하여 신규 화면으로 추가

### 클릭이벤트 적용
- 좌측 메뉴 > Screen > first 화면으로 이동
- first 화면에서 화면 터치시 main화면으로 이동하기
- 화면 중앙에 위차한 Block Rule 목록에서 Click 선택 
- New Rule 선택후 Node 부분에 "My First App"
- Click Action에 
```js filename="Click Action"
Jevil.go('main')
```
/ko/mobileapp/jevil/screenmove 참고
- Add New 클릭



![Quickstart 이미지](/image/click-mapping.png)
