# Devil App Builder 소개

**Devil App Builder **는 **Sketch** 또는 **Figma**에서 제작한 디자인 파일을 **실제 앱 화면으로 컨버팅**하는 개발 플랫폼입니다.

---

### 🔄 Sketch/Figma에서 만든 디자인 파일을 앱 화면으로 컨버팅
- 이 컨버팅은 1회성 작업이 아니라, **앱 개발과 운영 전 과정에서 디자인과 지속적으로 동기화**됩니다.  
- 즉, Sketch/Figma 디자인 파일을 수정하면 **앱에도 동기화**됩니다.

### 🛠️ 웹 콘솔 기반 개발
- 앱 개발은 **DAB 웹 콘솔**에서 진행합니다. 복잡한 앱개발 세팅이 필요없습니다.
- 개발된 결과물은 **DAB 전용 뷰어 앱**(Android & iOS)에서 컴파일없이 바로 확인할 수 있습니다.

### 🎨 UX & 로직 구조
- **UX/UI** → Sketch/Figma 디자인 + Block Rule  
- **앱 로직** → JavaScript로 개발  

### ⚡ 기능 업데이트는 앱 재심사 없이 온라인으로
- **컴파일 과정 없이 가볍고 빠르게 개발 가능**  
- **앱 심사 없이 새로운 기능을 즉시 추가 가능**

### 📱 앱 배포 방식
- 앱을 배포할 때 **DAB SDK를 탑재한 후** 를 받습니다. 이는 DAB 뷰어 앱과 기능이 일치합니다
- 출시 후에는 **기능 추가 시 다시 심사할 필요가 없이 기능이 업데이트 됩니다.**
